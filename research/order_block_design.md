Order Block Engine

Purpose:
Identify institutional order blocks generated by valid Break of Structure
(BOS) events.

The engine must be deterministic, objective, and reproducible.

Order Block Definition
Bullish Order Block

The final bearish candle immediately preceding the impulse that
creates a valid Bullish BOS.

Bearish Order Block

The final bullish candle immediately preceding the impulse that
creates a valid Bearish BOS.

Impulse

A contiguous directional price leg that

• begins immediately after the last confirmed opposite swing

• ends at the BOS candle

• contains no confirmed structural reversal

Open Questions

Q1. Order Block Search Boundary

Decision:

Search only within the impulse responsible for creating the valid BOS.

Search begins at the impulse start.

Search ends at the BOS candle.

Candles outside the impulse are ignored.


Q2. Impulse Start

Decision:

The impulse begins immediately after the latest confirmed pullback swing.

Bullish BOS

Impulse Start = HL

Impulse End = BOS

Bearish BOS

Impulse Start = LH

Impulse End = BOS

Q3. Order Block Selection

Decision:

Within the impulse responsible for creating the BOS,

the Order Block is the LAST opposite-colored candle.

Bullish BOS

→ Last bearish candle.

Bearish BOS

→ Last bullish candle.
